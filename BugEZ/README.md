# 💻 AI Code Analyzer (RAG using Endee Concept)

## 🚀 Overview
This project is an AI-powered code analyzer that explains code, detects errors, and suggests improvements.

## 🌐 Live Demo
👉 [Click here to use the app](https://bugezcodeanalyzer.streamlit.app/)

## 🔥 Features
- Code explanation  
- Time complexity analysis  
- Error detection  
- Optimization suggestions  
- Multi-language support  

## 🧠 Tech Stack
- Streamlit  
- Groq LLM  
- LangChain  
- Endee (Vector DB concept)  

## ⚙️ How it works
1. User inputs code  
2. Code is processed  
3. AI model analyzes it  
4. Generates explanation and suggestions  

## ▶️ Run Locally
```bash
pip install -r requirements.txt
